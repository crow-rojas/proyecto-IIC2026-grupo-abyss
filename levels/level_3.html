3. Codificación Visual y de Interacción

3.1. Elecciones de Marcas, Canales e Idioms Visuales

A continuación se muestran las elecciones de marcas, canales e idioms visuales para cada tarea visual:

Tarea Visual 1: Análisis de Tendencias

  • Marca: Líneas para representar tendencias de calificaciones a lo largo del tiempo.
  • Canal: Color para diferenciar entre animes; Eje X para el tiempo; Eje Y para calificaciones.
  • Idiom Visual: Gráfico de Multilínea, ya que permitirá una visualización clara de las tendencias de calificación a lo largo del tiempo.

Tarea Visual 2: Red de Recomendaciones

  • Marca: Nodos para animes y enlaces para recomendaciones.
  • Canal: Tamaño de nodos por popularidad; Color de enlaces por cantidad de votos favorables.
  • Idiom Visual: Diagrama Nodo Enlace, ya que es ideal para mostrar las interconexiones entre animes basadas en recomendaciones.

Tarea Visual 3: Análisis de animes atípicos

  • Marca: Áreas para representar diferentes géneros de animes atípicos.
  • Canal: Color para diferenciar géneros; Eje X para géneros; Eje Y para número de animes o calificaciones.
  • Idiom Visual: Treemap, ya que es eficaz para representar la distribución de géneros y subgéneros dentro del conjunto de datos.

3.2. Idioms de Interacción y Decisiones de Diseño

A continuación se muestran las decisiones de diseño para cada tarea visual:

  • Navegación y Selección: Implementación de d3.zoom para permitir a los usuarios explorar la Red de Recomendaciones con más detalle y uso de d3.brush en el Gráfico de Multilínea para seleccionar rangos de tiempo específicos.
  • Filtrado de Información: Inclusión de filtros de género y año en el Treemap mediante elementos de input HTML (<input/>). Esto permitirá a los usuarios enfocarse en animes específicos basados en su interés.
  • Interacción con el Mouse: Al pasar el mouse sobre un anime en cualquiera de las visualizaciones, se mostrará un tooltip con información detallada (nombre, calificación, género).
  • Vistas Coordinadas: Conexión entre el Gráfico de Multilínea y el Treemap: seleccionar un género en el Treemap actualizará el Gráfico de Multilínea para mostrar solo animes de ese género.
  • Uso de Data Joins: Se debe utilizar una aplicación correcta de Data Joins en D3.js para actualizar el Gráfico de Multilínea y la Red de Recomendaciones sin necesidad de recrear el SVG completo.
