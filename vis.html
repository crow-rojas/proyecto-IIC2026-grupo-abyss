Características del Gráfico

Este gráfico interactivo ofrece una variedad de características para una experiencia mejorada en la exploración de datos:

  • Desplazamiento Horizontal: Navegue a lo largo del eje x para ver conjuntos de datos extensos en detalle.
  • Información Detallada en Tooltips: Pase el cursor sobre los puntos de datos para mostrar información detallada en un tooltip.
  • Filtrado Flexible de Datos: Filtre los datos especificando un número de elementos aleatorios para mostrar o seleccionando un rango de fechas. La fecha incluirá años donde inicia o termina un anime.
  • Brush Interactivo: Utilice el click izquierdo para seleccionar y resaltar rangos de datos, y click derecho para restablecer la selección.

Encontrar animes atípicos

Cracterísticas del Gráfico

Este TreeMap interactivo está diseñado para proporcionar una visión detallada de los animes atípicos, aquellos que a pesar de tener un alto número de votos cuentan con una baja valoración. A través de esta visualización, se pueden explorar de forma intuitiva las siguientes interacciones:

  • Cambiar números de filtro: Se pueden cambiar los números de votos y rating para apreciar diferentes animes típicos o atípicos.

El objetivo de esta visualización es destacar los animes que, a pesar de su popularidad, podrían considerarse como infravalorados o que han generado división de opiniones entre los espectadores.
